# Closer 🎯

**Closer** is a smart iOS assistant that allows you to track your financial goals and works **directly integrated** with banking and stock market applications in Turkey.

Developed with SwiftUI, this application bridges financial apps with its user-friendly pastel interface and "Smart Deep Link" technology.

## ✨ Features

* **🎯 Goal Tracking:** Create separate goal cards for anything you want (Car, Vacation, Phone).
* **🔗 Smart Bank Integration:**
* Garanti BBVA, Akbank, İşCep, Binance, Midas, and more...
* Opens the app **directly** if it is installed.
* Redirects to the **App Store page** if it is not installed (Prevents region errors).


* **📊 Pastel Analysis Dashboard:** Monitor your total savings status and progress with stylish charts.
* **🎨 Customization:** Choose different color themes and icons for each goal.
* **📱 Modern UI:** Written in SwiftUI, providing a fluid interface compliant with Apple's design language.

## 🚀 Technologies

* **Language:** Swift 5
* **Framework:** SwiftUI
* **Storage:** @AppStorage / UserDefaults
* **Architecture:** Data flow compliant with the MVVM (Model-View-ViewModel) structure.
* **Other:** Universal Links, URL Schemes, Custom URL Handling.
